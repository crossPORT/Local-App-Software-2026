RocketBox — USB fabric file transfer
=====================================

Desktop (wx)
  Launch RocketBox from the Start Menu / Applications / installed menu entry.
  Connect the SLS fabric USB cable before opening the app.

  Linux USB access (one-time):
    sudo cp /usr/share/rocketbox/99-sls-fabric-usb.rules /etc/udev/rules.d/
    sudo udevadm control --reload-rules && sudo udevadm trigger
    Unplug and replug the cable.
